# Smart-email-assistance
Smart Email Assistant : Spring Boot + Gemini AI + Chrome Extension (Gmail Integration). 

AI-powered email reply generator integrated directly inside Gmail.

This project is a full-stack Smart Email Assistant that helps users compose email replies automatically using AI.
It integrates with Gmail's compose window, adds an “AI Reply” button, and fetches an AI-generated professional email response from a Spring Boot backend powered by Google Gemini AI.
